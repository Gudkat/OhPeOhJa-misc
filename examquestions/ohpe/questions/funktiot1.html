Funktiot tietotyyppinä

Kurssilla on käsitelty tietotyypejä kuten merkkijonot ja kokonaisluvut. Myös funktiot ovat oma tietotyyppinsä. Tämä tarkoittaa, että funktioita voidaan käsitellä aivan kuten muitakin tietotyyppejä. Niitä voidaan sijoittaa muuttujiin, antaa parametrinä toiselle funktiolle ja niin edespäin.

Alla oleva koodinpätkä antaa pari konkreettista esimerkkiä:

# Määritellään funktio
def testifunktio():
    print('Terveiset testifuntiosta')

# Luodaan funktio, joka saa parametrina toisen funktion
def aja_funktio(funktio):
    # Kutsutaan parametrina annettua funktiota aja_funktio:n sisällä
    funktio()

# Nyt aja_funktion sisällä kutsutaan testifunktiota 
aja_funktio(testifunktio) # Tulostuu 'Terveiset testifunktiosta'

Luo funktio muunna(lista, funktio), joka toimii seuraavasti:

muunna-funktio syöttää parametrina annetun listan jokaisen alkion parametrinä annetulle funktiolle. Funktiolta saadut arvot lisätään uuteen listaan, jonka muunna-funktio lopulta palauttaa. Voit olettaa, että annettu lista ja funktio ovat sopivanlaisia. Alla esimerkki muunna-funktion käytöstä.

def lisaa_yksi(luku):
    return luku + 1

lista = [1,1,1]

lisatty = muunna(lista, lisaa_yksi)
print(lisatty) # Tulostaa: [2,2,2]
