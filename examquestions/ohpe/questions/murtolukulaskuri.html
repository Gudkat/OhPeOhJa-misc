Kopioi alla oleva runko tehtäväpohjaan.

from fractions import Fraction

def murtolukulaskuri(lasku: str):
    pass

def muunna_murtoluvuksi(murtoluku: str):
    pass

Fraction-olio toimii siten, että se saa kaksi arvoa; ensimmäinen arvo on osoittaja ja toinen arvo on nimittäjä. Esimerkiksi puolikasta vastaava murtoluku luodaan Fraction(1,2). Murtolukuja voi luoda myös desimaaleista Fraction-olion avulla myös, mutta sitä ei tässä tehtävässä tarvita.

Tehtävänä on ohjelmoida murtolukulaskin, jolla voi suorittaa seuraavat operaatiot:

  • Yhteenlasku
  • Vähennyslasku
  • Kertolasku
  • Murtoluvun supistus, jonka Fraction-olio tekee automaattisesti.

Rakenna ohjelma siten, että funktio murtolukulaskuri kutsuu muunna_murtoluvuksi funktiota. murtolukulaskurille annetaan arvoksi laskutoimitus merkkijonona ja se palauttaa laskutoimituksen tuloksen merkkijonona. muunna_murtoluvuksi funktio saa arvoksi murtoluvun merkkijonona, ja palauttaa Fraction-olion.

Esimerkki muunna_murtoluvuksi funktion toiminnasta:

murtoluku1 = "1/2"
murtoluku2 = "15/375"
murtolukuolio = muunna_murtoluvuksi(murtoluku2)
print(muunna_murtoluvuksi(murtoluku1))
print(type(murtolukuolio))
print(murtolukuolio)

Esimerkkitulostus:

1/2
<class 'fractions.Fraction'>
1/25

Esimerkki murtolukulaskuri funktion toiminnasta:

lasku1 = "1/2 + 3/4"
lasku2 = "1/2 - 1/3"
lasku3 = "1/2 * 1/4"
supistettava = "15/375"

summa = murtolukulaskuri(lasku1)
erotus = murtolukulaskuri(lasku2)
tulo = murtolukulaskuri(lasku3)
supistettu = murtolukulaskuri(supistettava)

print("supistettavan tyyppi:", type(supistettava))
print("murtoluvun tyyppi:", type(muunna_murtoluvuksi(supistettava)))
print("supistetun tyyppi:",type(supistettu))

print(f'laskun {lasku1} summa on', summa)
print(f'laskun {lasku2} erotus on', erotus)
print(f'laskun {lasku3} tulo on', tulo)
print(f'murtoluvun {supistettava} supistettu muoto on', supistettu)

# Lasketaan (1/2 + 3/4) * (1/2 - 1/3) käyttäen hyväksi aiempien laskujen palautuksia
lasku4 = f"{summa} * {erotus}"
print(murtolukulaskuri(lasku4))

Esimerkkitulostus:

murtoluvun tyyppi: <class 'fractions.Fraction'>
supistetun tyyppi: <class 'str'>
laskun 1/2 + 3/4 summa on 5/4
laskun 1/2 - 1/3 erotus on 1/6
laskun 1/2 * 1/4 tulo on 1/8
murtoluvun 15/375 supistettu muoto on 1/25
