Kurssilla on käsitelty tietotyypejä kuten merkkijonot ja kokonaisluvut. Myös funktiot ovat oma tietotyyppinsä. Tämä tarkoittaa, että funktioita voidaan käsitellä aivan kuten muitakin tietotyyppejä. Niitä voidaan sijoittaa muuttujiin, antaa parametrinä toiselle funktiolle ja niin edespäin.

Alla oleva koodinpätkä antaa konkreettisen esimerkin:

# Määritellään funktio
def testi_funktio():
    print('Terveiset testifunktiosta')

# Luodaan funktio, joka saa parametrina toisen funktion
def suorita_funktio(funktio):
    # Kutsutaan parametrina annettua funktiota
    funktio()

# Syötetään testi_funktio suorita_funktio:lle
suorita_funktio(testi_funktio) # Tulostuu: 'Terveiset testifunktiosta'

Luo tehtäväpohjaan kaksi funktiota. Funktiot on määritelty seuraavanlaisesti:

a) def suorita_monesti(funktio, n: int), joka suorittaa parametrina annetun funktion n kertaa.

Voit kokeilla koodiasi alla olevalla esimerkillä:

# testi_funktio määritelty ylempänä
suorita_monesti(testi_funktio, 3) # Tulostaa 'Terveiset testifunktiosta' 3 kertaa
print()
suorita_monesti(testi_funktio, 5) # Tulostaa 'Terveiset testifunktiosta' 5 kertaa

b) def tarkista_lista(tarkistus_funktio, lista: list), joka ajaa tarkistusfunktion parametrina annetun listan jokaiselle alkiolle. Jos tarkistus_funktio palauttaa True jokaiselle listan alkiolle, niin tarkista_lista palauttaa True. Muutoin tarkista_lista palauttaa False. Voit olettaa, että tarkistus_funktio on yksipaikkainen funktio, joka palauttaa totuusarvon.

Voit kokeilla koodiasi alla olevalla esimerkillä:

def suurempi_kuin_nolla(luku: int):
    return luku > 0

def on_palindromi(merkkijono: str): 
    return merkkijono == merkkijono[::-1]

print(tarkista_lista(suurempi_kuin_nolla, [1,2,3])) # True
print(tarkista_lista(suurempi_kuin_nolla, [1,2,0])) # False
print(tarkista_lista(on_palindromi, ['abba', 'saippuakauppias'])) # True
print(tarkista_lista(on_palindromi, ['abba', 'python'])) # False
