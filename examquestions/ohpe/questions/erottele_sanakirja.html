Tässä tehtävässä luodaan kaksi funktiota tehtäväpohjaan. Funktiot on määritelty seuraavanlaisesti:

a) def erottele(merkkijono: str)

Palauttaa listan tupleja parametrina annetun merkkijonon perusteella. Listan tuplet ovat kaksipaikkaisia. Merkkijonot ovat muotoa "jono1a;jono1b,jono2a;jono2b,jono3a; ...". Tuplet erotellaan merkkijonosta pilkun perusteella ja tuplen osat puolipisteellä, niin että puolipisteen vasemmanpuolinen merkkijono on tuplen vasemmanpuolinen alkio.

Kokeile koodisi toimintaa alla olevalla esimerkillä.

lista1 = erottele('maa;suomi,kaupunki;helsinki,huone;keittio')
lista2 = erottele('nimi;ella')
lista3 = erottele('valmistaja;aston martin,kuski;vettel')

print(lista1) # [('maa', 'suomi'), ('kaupunki', 'helsinki'), ('huone','keittio')]
print(lista2) # [('nimi', 'ella')]
print(lista3) # [('valmistaja', 'aston martin'), ('kuski', 'vettel')]

b) def sanakirjaksi(lista: list)

Palauttaa sanakirjan parametrina annetun listan pohjalta. Annettu lista koostuu kaksipaikkaisista tupleista, kuten a-kohdan funktion paluuarvo. Sanakirja rakennetaan niin, että listan tuplet vastaavat avain-arvo-pareja, missä tuplen vasen alkio on avain ja oikea alkio on arvo.

Kokeile koodisi toimintaa alla olevalla esimerkillä.

sanakirja1 = sanakirjaksi([('maa', 'suomi'), ('kaupunki', 'helsinki'), ('huone','keittio')])
sanakirja2 = sanakirjaksi([('nimi', 'ella')])
sanakirja3 = sanakirjaksi([('valmistaja', 'aston martin'), ('kuski', 'vettel')])

print(sanakirja1) # {'maa': 'suomi', 'kaupunki': 'helsinki', 'huone': 'keittio'}
print(sanakirja2) # {'nimi': 'ella'}
print(sanakirja3) # {'valmistaja': 'aston martin', 'kuski': 'vettel'}
